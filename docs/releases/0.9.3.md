# Videorc 0.9.3 Beta 1

Videorc 0.9.3 is the **black glass** release: the logo-matched theme in both
modes, user-controlled background visibility, contain-never-crop screens, and
the preview first-frame contract.

## Scope

- **Black glass theme** — dark = black glass (tokens L0.13/68%, dimmed frost,
  polished hairlines, specular shine utility), light = porcelain twin; brand red
  unified to the logo; main-side windows on `window-palette.ts`.
- **Background Visibility slider** (0–40%, default 20 = classic 80% stage) with
  per-asset defaults + per-scene overrides; **Remove from scene** returns the
  recording to the full canvas.
- **Screens contain, never crop** in every renderer + the recording graph
  (transparent letterbox pads; Dock always visible).
- **Preview first-frame contract** — self-healing ladder + truthful waiting
  hints; real-user launch smoke folded into the studio gates; backend state
  isolation for smokes (the profile-pollution fix).
- **WebP/JPEG background decode** + non-fatal background validation at commit.
- Everything in 0.9.2 (capture entitlements fix, in-app updater).
